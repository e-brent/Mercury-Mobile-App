# Mercury-Mobile-App
A mobile app for the camera company Mercury, containing calculators and the user guide for the stereo camera system

## Code files summary
**App.js** - *Contains the navigation stack for the app and the shared header for the other pages.*
**home.js** - *The home screen and initial render of the app. Contains buttons for navigating to all other screens.*
**dofCalculator** - *Contains the Depth of Field and Hyperfocal calculators, which can be toggled between.*
**reciprocityCalculator** - *A calculator for the reciprocity of different film types.*
**baseCalculator** - *Contains the calculator for the distance between two lens bases for stereo photography.*
